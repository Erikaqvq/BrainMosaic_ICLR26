## Public Data Templates

This folder provides minimal public templates for external users who want to prepare
custom BrainMosaic text-side and EEG-side inputs.

### Files

- `sentences.template.csv`
  Source sentences used to build `sentence_embeddings.pt`.

- `token_explanations.template.json`
  Token vocabulary plus optional short explanations used to build `word_embeddings.pt`.

- `segmentation.template.json`
  Optional sentence-to-token mapping plus optional sentence-level labels for `segmentation_path`.
